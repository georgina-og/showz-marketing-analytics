# 📊 Marketing Analytics & User Behavior — Showz

## 🎯 Objetivo
Análisis end-to-end del comportamiento de usuarios de Showz (plataforma de venta de entradas)
para optimizar el ROMI por canal y período de inversión.

## 🛠️ Herramientas
Python · Pandas · NumPy · Seaborn · Matplotlib

## 📌 Métricas construidas
- **Producto:** DAU/WAU/MAU · Sticky factor · ASL
- **Negocio:** Tiempo a primera compra · Tasa de conversión por cohorte · LTV acumulativo
- **Marketing:** CAC y ROMI por fuente (10 canales) · Segmentación desktop vs. touch

## 💡 Hallazgos clave
- 84% de usuarios no convirtieron; el 11.5% compró el mismo día de su primera visita
- Cohortes Sep–Dic 2017: mayor LTV acumulado (hasta $34,676) y mayor recurrencia
- Fuente 1: mayor ROMI del portafolio · Fuente 3: mayor CAC, menor retorno
- Touch devices: menor volumen pero ticket promedio 17% superior al de desktop

## 🎯 Impacto
Recomendaciones para redirigir la inversión publicitaria hacia los canales y períodos
con mayor ROMI, con base en análisis de cohortes y LTV acumulativo.
